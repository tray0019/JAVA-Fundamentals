# Java Fundamentals

A collection of beginner to advance level Java programs demonstrating core concepts.

# How to run
1. Clone the repo
2. Open in Java IDE
3. Compile and run

# Features
- Basic Syntax and Structures
- Data types
- Conditional Statements
- Loops and methods
- Exceptions
- OOP concepts: classes, objects, inheritance, encapsulation
- Data Structures

# Technologies
- Java 21
- Git & GitHub
- Eclipse
